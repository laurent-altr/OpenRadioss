# Model Readers (`reader/source/io/model_readers/`)

Format-specific model reader implementations for Radioss and LS-DYNA decks, plus shared utility routines.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `radioss/` | Radioss format reader (`.rad` / `_0001.rad`) |
| `ls-dyna/` | LS-DYNA keyword format reader |
| `util/` | Shared reader utility functions |
