# Tools (`tools/`)

Standalone post-processing utilities and development mockups for OpenRadioss.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `anim_to_vtk/` | Converts OpenRadioss animation (`.anim`) files to legacy VTK ASCII format |
| `th_to_csv/` | Converts OpenRadioss time-history (`.th`) files to CSV format |
| `mockup/` | Minimal standalone test harnesses for contact detection and element computation |
