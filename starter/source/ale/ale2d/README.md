# ALE 2D Initialisation (`starter/source/ale/ale2d/`)

Starter initialisation routines for 2D ALE (plane-strain/axisymmetric) mesh and material setup.

## Key Files

| File | Role |
|------|------|
| `agrad2.F` | 2D ALE gradient computation for initial mesh |
| `aleso2.F` | 2D ALE solid element setup |
| `aleso2t.F` | 2D ALE solid element setup with temperature |
| `aturi2.F` | 2D ALE turbulence initialisation |
| `edlen2.F` | 2D ALE element edge length computation |
