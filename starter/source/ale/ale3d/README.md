# ALE 3D Initialisation (`starter/source/ale/ale3d/`)

Starter initialisation routines for 3D ALE mesh and material setup.

## Key Files

| File | Role |
|------|------|
| `agrad3.F` | 3D ALE gradient computation for initial mesh |
| `aleso3.F` | 3D ALE solid element setup |
| `aturi3.F` | 3D ALE turbulence initialisation |
| `edlen3.F` | 3D ALE element edge length computation |
| `enorm3.F` | 3D ALE element face normal computation |
| `eporin3.F` | 3D ALE element orientation |
| `iface.F` | ALE element face connectivity setup |
