## Model Analysis (`analyse/`)

The `analyse/` subdirectory performs sanity checks on the assembled model:

| File | Role |
|------|------|
| `analyse.c` | Main analysis dispatcher |
| `analyse_GUI.c` | GUI-compatible analysis output |
| `analyse_arret.F` | Fatal error during analysis |
| `analyse_char.F` | Character-based analysis output |
| `analyse_check.c` | Specific validity checks (aspect ratios, volumes, etc.) |

These checks include:
- Negative-volume elements detection
- Free edge / free surface identification
- Mass and inertia sanity
- Contact interface validity (overlapping surfaces, zero-area segments)

## Initial Animation (`anim/`)

Writes the initial geometry (t=0) to the animation file before the engine starts. This gives HyperView users a reference state.

## QA Print (`qaprint/`)

The `st_qaprint_*.F` files write the per-keyword parameter tables to the `.out` file when `/QAPRINT` is active. There is one file per major keyword family:

| File prefix | Keyword family |
|-------------|---------------|
| `st_qaprint_admesh` | Adaptive meshing |
| `st_qaprint_ale_options_driver` | ALE options |
| `st_qaprint_clusters` | Cluster regions |
| `st_qaprint_composite_options` | Composite properties |
| `st_qaprint_constraints` | Kinematic constraints |
| ‚Ä¶ | (many more, one per keyword family) |

## Output File Sections (`outfile/`)

| File | Output section |
|------|---------------|
| `titre1.F`, `titre2.F`, `titre3.F` | Header title blocks in `.out` |
| `printgroup.F` | Print group (set) membership summary |

## Statistics (`stat/`)

Writes the model statistics block at the end of the starter run:
- Total number of nodes, elements by type
- Total model mass
- Initial kinetic energy (from `/INIVEL`)
- Minimum element size (critical for expected DT)
