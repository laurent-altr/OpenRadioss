# General Loads (`starter/source/loads/general/`)

Reads all general structural load types: concentrated forces, gravity, centrifugal, cylindrical pressure, surface pressure, and fluid pressure.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `cload/` | Concentrated nodal forces and moments |
| `grav/` | Gravity body force |
| `load_centri/` | Centrifugal and Coriolis loads |
| `load_pcyl/` | Cylindrical coordinate pressure load |
| `load_pressure/` | Surface pressure loads |
| `pfluid/` | Fluid pressure on structure |
| `pload/` | General pressure load |
| `preload/` | Preload conditions |
