# Solid Connector Initialisation (`starter/source/elements/solid/sconnect/`)

Starter initialisation for solid connector elements (zero-length or short cohesive solid connectors).

## Key Files

| File | Role |
|------|------|
| `sini43.F` | Initialise 4/3-node solid connector |
| `spcoor3.F` | Compute solid connector coordinate system |
| `lce16q.F` | Initialise 16-node quadratic connector |
