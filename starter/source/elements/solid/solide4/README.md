# Tet4 Solid Initialisation (`starter/source/elements/solid/solide4/`)

Starter initialisation for 4-node tetrahedral solid elements.

## Key Files

| File | Role |
|------|------|
| `s4init3.F` | Main tet4 element initialisation |
| `s4mass3.F` | Compute tet4 nodal mass |
| `s4coor3.F` | Compute tet4 coordinate system |
| `s4deri3.F` | Compute tet4 shape function derivatives |
| `s4defo3.F` | Compute tet4 deformation gradient |
| `s4jaci3.F` | Compute tet4 Jacobian |
| `s4orth3.F` | Compute tet4 orthotropic axes |
| `s4refsta3.F` | Set tet4 reference state |
| `s4repiso3.F` | Tet4 isotropic stress representation |
| `s4rota3.F` | Tet4 rotation initialisation |
