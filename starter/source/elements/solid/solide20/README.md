# 20-Node Solid Initialisation (`starter/source/elements/solid/solide20/`)

Starter initialisation for 20-node serendipity hexahedral solid elements.

## Key Files

| File | Role |
|------|------|
| `s20init3.F` | Main 20-node element initialisation |
| `s20mass3.F` | Compute 20-node solid nodal mass |
| `s20coor3.F` | Compute element coordinate system |
| `s20deri3.F` | Compute shape function derivatives |
| `s20temp.F90` | Initialise temperature field for 20-node elements |
