# 8-Node Z-Strain Solid Initialisation (`starter/source/elements/solid/solide8z/`)

Starter initialisation for 8-node hexahedral solid elements with Z-type strain formulation (TYPE24).

## Key Files

| File | Role |
|------|------|
| `s8zinit3.F` | Main Z-strain solid initialisation |
| `s8zderi3.F` | Compute Z-strain shape function derivatives |
| `s8erefcoor3.F` | Set element reference coordinates |
