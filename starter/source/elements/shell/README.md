# Shell Element Initialisation (`starter/source/elements/shell/`)

Starter initialisation for 4-node shell elements: QEPH (coque), Belytschko-Tsay (coqueba), and shell offset geometry.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `coque/` | QEPH 4-node shell element initialisation |
| `coqueba/` | Belytschko-Tsay (BT) 4-node shell initialisation |
| `shell_offset/` | Shell mid-surface offset geometry initialisation |

## Key Files (top-level)

| File | Role |
|------|------|
| `get_sort_key_shell.F90` | Compute sort key for shell element ordering |
