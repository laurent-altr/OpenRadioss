# Wedge Thick-Shell Initialisation (`starter/source/elements/thickshell/solide6c/`)

Starter initialisation for 6-node pentahedral (wedge) solid-shell elements.

## Key Files

| File | Role |
|------|------|
| `s6cinit3.F` | Main wedge thick-shell initialisation |
| `s6ccoor3.F` | Compute wedge thick-shell coordinate system |
| `s6cderi3.F` | Compute wedge shape function derivatives |
| `s6cortho3.F` | Compute wedge orthotropic axes |
| `s6fraca3.F` | Wedge fracture area computation |
| `s6mass3.F` | Compute wedge thick-shell nodal mass |
