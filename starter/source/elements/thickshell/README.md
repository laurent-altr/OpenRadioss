# Thick-Shell Elements (`starter/source/elements/thickshell/`)

Starter initialisation for thick-shell (solid-shell) element formulations: TYPE1, TYPE2 (ANS+EAS), wedge, and 16-node quadratic.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `solidec/` | TYPE1 thick-shell (8-node, QEPH) |
| `solide8c/` | TYPE2 thick-shell (8-node, ANS+EAS, implicit-ready) |
| `solide6c/` | Wedge (6-node) thick-shell |
| `solide16/` | 16-node quadratic thick-shell |
