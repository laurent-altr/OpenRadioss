# 16-Node Thick-Shell Initialisation (`starter/source/elements/thickshell/solide16/`)

Starter initialisation for 16-node quadratic solid-shell elements.

## Key Files

| File | Role |
|------|------|
| `s16init3.F` | Main 16-node thick-shell initialisation |
| `s16mass3.F` | Compute 16-node thick-shell nodal mass |
| `s16coor3.F` | Compute 16-node element coordinate system |
| `s16deri3.F` | Compute 16-node shape function derivatives |
