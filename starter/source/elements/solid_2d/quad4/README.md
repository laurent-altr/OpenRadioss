# 2D Quad4 EAS Element Initialisation (`starter/source/elements/solid_2d/quad4/`)

Starter initialisation for 4-node quadrilateral 2D solid elements with Enhanced Assumed Strain (EAS).

## Key Files

| File | Role |
|------|------|
| `q4init2.F` | Main 2D EAS quad element initialisation |
| `q4coor2.F` | Compute EAS quad coordinate system |
| `q4deri2.F` | Compute EAS quad shape function derivatives |
| `q4voli2.F` | Compute EAS quad element area |
