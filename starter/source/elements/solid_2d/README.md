# 2D Solid Elements (`starter/source/elements/solid_2d/`)

Starter initialisation for 2D solid elements: quadrilateral and triangular plane-strain / axisymmetric elements.

## Subdirectories

| Directory | Contents |
|-----------|---------|
| `quad/` | 4-node quadrilateral 2D solid (one-point integration) |
| `quad4/` | 4-node quadrilateral with enhanced strain (EAS) |
| `tria/` | 3-node triangular 2D solid |
