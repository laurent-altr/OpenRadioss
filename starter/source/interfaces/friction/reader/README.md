# Friction Reader (`starter/source/interfaces/friction/reader/`)

Alternate reader location for friction model cards, providing the same parsing routines as the parent friction directory.

## Key Files

| File | Role |
|------|------|
| `hm_read_friction.F` | Parse /FRICTION card |
| `hm_read_friction_models.F` | Parse named friction model parameters |
| `hm_read_friction_orientations.F` | Parse orthotropic friction directions |
