# Starter LAW40 — Mooney-Rivlin hyperelastic (`starter/source/materials/mat/mat040/`)

Reads `/MAT/LAW40` keyword parameters from the Radioss deck and stores
them in the `PM` / `IPM` arrays passed to the engine material routine.

| File | Role |
|------|------|
| `hm_read_mat40.F` | `HM_READ_MAT40` — reads `/MAT/LAW40` keyword parameters into PM/IPM |
