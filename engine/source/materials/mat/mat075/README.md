# LAW75 — Visco-Elastic (Kelvin-Voigt) (`engine/source/materials/mat/mat075/`)

Kelvin-Voigt visco-elastic material: spring and dashpot in parallel,
giving stress σ = E ε + η ε̇. Suitable for polymers and biological tissue.
