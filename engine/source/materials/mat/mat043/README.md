# LAW43 — Anisotropic Elastic-Plastic (Hill 48) (`engine/source/materials/mat/mat043/`)

Elastic-plastic material using the Hill 1948 quadratic yield surface with
tabulated isotropic hardening; full 3D solid and shell support.
