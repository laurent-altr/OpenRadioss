# LAW81 — Elastic-Plastic with Thermal Softening (`engine/source/materials/mat/mat081/`)

Elastic-plastic material with adiabatic thermal softening from plastic
dissipation; yield stress decays with temperature rise T = T₀ + βW_p/ρCₚ.
