# LAW21 — Elastic-Plastic with EOS (Mie-Grüneisen) (`engine/source/materials/mat/mat021/`)

Elastic-plastic material with Mie-Grüneisen equation of state for high-pressure
applications. Combines radial-return plasticity with shock EOS pressure.
