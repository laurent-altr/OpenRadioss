# LAW124 — Hyperelastic (Gent) (`engine/source/materials/mat/mat124/`)

Gent hyperelastic model for rubber: W = −μJ_m/2 ln(1−(Ī₁−3)/J_m).
Captures finite extensibility via the limiting stretch J_m.
