# LAW28 — Hyperelastic Mooney-Rivlin (`engine/source/materials/mat/mat028/`)

Incompressible Mooney-Rivlin hyperelastic material for rubber and elastomers.
W = C₁(I₁−3) + C₂(I₂−3). Large-strain total-Lagrangian formulation.
