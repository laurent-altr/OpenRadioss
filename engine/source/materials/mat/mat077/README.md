# LAW77 — User Material (VUMAT Interface) (`engine/source/materials/mat/mat077/`)

Interface to user-defined material subroutines in Abaqus VUMAT format,
allowing existing VUMAT subroutines to be used with OpenRadioss.

## Key Files

| File | Role |
|------|------|
| `sigeps77.F` | VUMAT call-out wrapper: converts Radioss stress state to VUMAT convention |
