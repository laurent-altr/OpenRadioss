# LAW50 — Visco-Hyperelastic Mooney (`engine/source/materials/mat/mat050/`)

Mooney-Rivlin hyperelastic material with Prony-series visco-elastic
correction; used for rate-dependent rubber and elastomers.

## Key Files

No Fortran `.F` files (uses shared hyperelastic kernel from `mat_share/`).
