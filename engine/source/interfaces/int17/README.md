# TYPE17 Interface (`engine/source/interfaces/int17/`)

TYPE17 — General segment-to-segment contact with edge contact (similar to TYPE25 but older formulation).

## Key Files

| File | Role |
|------|------|
| `i17buce.F` | Bucket cell computation |
| `i17crit.F` | Penetration criterion |
| `i17for3.F` | Contact force computation |
