About OpenRadioss

OpenRadioss is an open-source finite element analysis (FEA) solver designed for structural mechanics simulations. It provides powerful capabilities for crash simulation, impact analysis, and dynamic structural analysis.

Originally developed as a commercial solver, OpenRadioss has been released as open source to benefit the engineering and research community worldwide.
